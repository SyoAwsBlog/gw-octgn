これは、OCTGN3 ガンダムウォー（旧版） ゲーム定義ソース コードの github リポジトリです。

## プレイヤー ガイド ##

カードデータは、旧版（Octgn 1.3.529用に作られたガンダムウォーパッチのインターネットアーカイブ）から取得して利用する前提となっています。
旧版の在り処は、ググってください。

注: OCTGN クライアント自体に関連する問題は報告しないでください。http://www.github.com/octgn/OCTGN/issues で報告する必要があります。
